Introduction

PDF Query Assistant is a powerful tool that allows you to extract information from PDF documents using natural language queries. Built with FastAPI and Ollama's Mistral model, it provides intelligent responses based solely on the content of your uploaded PDF files.

This application is designed for professionals, researchers, students, and anyone who needs to quickly find information within PDF documents without manually scanning through pages of content.

Features

Core Functionality

  • Upload PDF documents via drag-and-drop or file browser
  • Ask natural language questions about your PDF content
  • Receive accurate answers based solely on the document content
  • Clean, professional interface with dark mode support
  • Responsive design that works on desktop and mobile devices

Advanced Capabilities

  • Extracts text from complex PDF layouts
  • Handles multi-page documents efficiently
  • Maintains context from previous questions in a session
  • Formats responses clearly with relevant document sections
  • Provides citations to specific parts of the document when possible

How It Works

Architecture

The PDF Query Assistant follows a client-server architecture:

  1. Frontend: A responsive web interface built with HTML, CSS, and JavaScript
  2. Backend: A FastAPI server that handles file uploads and processes requests
  3. AI Model: Ollama with the Mistral instruct model for natural language processing

Processing Pipeline

1

PDF text extraction using PyMuPDF

2

Query formulation with strict instructions to use only PDF content

3

AI processing via Ollama's API

4

Response delivery to the user interface

Note: The AI model is specifically instructed to only use information present in the uploaded PDF. If the answer cannot be found in the document, it will respond with "Not mentioned in the PDF."

Troubleshooting

Common Issues

PDF upload fails

  • Ensure the file is a valid PDF document
  • Check that the file is not password protected
  • Verify the file size is reasonable (under 100MB recommended)

No response from the AI model

  • Ensure Ollama is running and the Mistral model is installed
  • Check your network connection
  • Verify the server logs for any errors

Inaccurate answers

  • The AI model only uses information present in the PDF
  • Try rephrasing your question for better results
  • For complex documents, ask more specific questions
Important: This tool extracts text from PDFs but may not perfectly handle complex layouts, scanned documents (images), or heavily formatted content. For best results, use text-based PDFs rather than scanned documents.
